# Projeto-Bootcamp-Santander-CyberSeguran-a---DIO
Projeto de cibersegurança realizado em ambiente controlado usando Kali Linux e Metasploitable. Inclui enumeração, brute force e password spraying para demonstrar riscos reais de credenciais fracas e validar superfícies de autenticação, reforçando práticas de hardening e segurança corporativa.
